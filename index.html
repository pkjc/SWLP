Semantic Web Based Learning Platform


SWLP creates a more comprehensive online learning environment.
Here's how it works-

  • Students/teachers add appropriate metatags (topics) to parts of lecture videos
  • SWLP finds resources from all over the web that are semantically related to the metatags so that students can explore each topic/sub-topic in more detail.
